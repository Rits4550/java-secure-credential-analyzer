# Secure Credential Analyzer (Java OOP)

## Overview
A Java-based console application that analyzes password strength and identifies basic security risks using Object-Oriented Programming principles.

## Features
- Password length validation
- Complexity analysis (uppercase, lowercase, digits)
- Detection of commonly used weak passwords
- Modular and extensible OOP design

## Concepts Used
- Interfaces and abstraction
- Inheritance and polymorphism
- Encapsulation
- Secure coding awareness

## What I Learned
- Designing extensible Java applications using OOP
- Applying security concepts to real-world scenarios
- Writing clean, modular, and maintainable code
- Rapidly learning and implementing new design patterns

## How to Run
javac Main.java  
java Main
